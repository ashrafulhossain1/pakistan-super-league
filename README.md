# IPL Website: Your Ultimate Cricket Hub


## Description: 
The IPL Hub Website is an interactive platform where you can engage with the IPL experience by claiming free money through a dedicated button, selecting your favorite players, and managing your team by adding or deleting players as needed. Each action on the site triggers alerts for a user-friendly experience, and the website is designed with an impressive footer that completes the overall look and feel. Enjoy exploring various features, managing players, and staying connected with the IPL excitement!


## Technologies Used

- **HTML5**: Used for structuring the basic layout of web pages, enabling browsers to display content.

- **CSS**: Utilized for styling and layout design of web pages, enhancing the user experience.

- **Tailwind CSS**: A utility-first CSS framework that allows for rapid and efficient custom design creation.

- **DaisyUI**: A UI library built on top of Tailwind CSS, providing pre-designed components for faster development.

- **JavaScript**: A programming language used to add interactivity to web pages.

- **React**: A popular JavaScript library for building user interfaces, supporting component-based development.


## Key Features

- **Free Money Claim**: Users can easily claim free money by clicking a button, enhancing their ability to build and manage their team without financial constraints.

- **Player Selection and Management**: Users can select their favorite players to form a team, with the ability to add and remove players as needed, allowing for flexible team management.

- **Real-Time Alerts**: The website provides alerts for various actions, ensuring users are informed about important updates, such as successful player additions or changes in their team.
